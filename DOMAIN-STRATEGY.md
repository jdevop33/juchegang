## Content Routing Rules

```
IF content is about:
├── Korean history/culture/reunification → juche.org
├── Geopolitics/sanctions/media analysis → juche.org/briefings
├── Debt/markets/economic data → 1929.world
├── Crypto/TON/tokens → TON project sites
├── Fashion/products → outlierclothiers.com
└── Professional/career → jesse-resume
```

---

## Cross-Linking (DO)

✅ juche.org briefing about sanctions → link to 1929.world for economic data
✅ 1929.world debt analysis → link to juche.org for geopolitical context
✅ All sites → link to jesse-resume in subtle footer attribution

## Cross-Linking (DON'T)

❌ Don't put crypto content on juche.org
❌ Don't put peace activism on 1929.world
❌ Don't put commercial products on any non-commercial site
❌ Don't mix personal opinions with jesse-resume
