# Juche.org Brand Kit

## Mission Statement
Uniting Korea through pride, shared heritage, and mutual respect. Celebrating the resilience and achievements of the Korean people - North and South - while advocating for peace and self-determination.

## Core Message
**"One Korea, One People, One Pride"**

The 38th parallel created an experiment that revealed the extraordinary capability of the Korean people under vastly different circumstances. Both succeeded against the odds. Time to celebrate that together.

## Visual Elements

### Gradients
```css
/* Hero backgrounds */
background: linear-gradient(to bottom, #000000, #111827, #000000);

/* Card accents */
background: linear-gradient(to right, rgba(3, 77, 162, 0.2), rgba(139, 92, 246, 0.2), rgba(3, 77, 162, 0.2));

/* Featured sections */
background: linear-gradient(to right, rgba(236, 29, 37, 0.1), rgba(3, 77, 162, 0.1));
```

### Border Styles
- Standard: 1px solid rgba(255, 255, 255, 0.1)
- Accent: 1px solid [color]/30 (30% opacity of accent color)
- Focus: 2px solid Unity Blue

### Card Styling
```css
background: rgba(0, 0, 0, 0.3);
backdrop-filter: blur(12px);
border-radius: 0.75rem;
border: 1px solid rgba(255, 255, 255, 0.1);
```

### Shadow
```css
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);
```

## Logo Concepts

### Primary Mark Options
1. **Tiger Unity** - Two tigers (symbols of Korea) facing each other, forming a circle
2. **Peninsula Embrace** - Stylized Korean peninsula with unity gradient
3. **48 Star** - Eight-pointed star representing the 48 Laws of Excellence
4. **Friendship Flame** - Torch with blue and red intertwined flames

### Logo Usage
- Minimum size: 32px height
- Clear space: 1x logo height on all sides
- On dark backgrounds: Full color or white
- On light backgrounds: Full color or black

## Photography & Imagery

### Style Guidelines
- Documentary/authentic over staged
- Warm color grading
- Human-focused: real people, real moments
- Cultural elements: traditional and modern Korea
- Landscapes: Korean peninsula beauty
- Avoid: propaganda aesthetics, militaristic imagery

### Image Treatment
- Subtle vignette on edges
- Color grade toward warm tones
- Overlay gradients for text legibility

## Voice & Tone

### Principles
1. **Factual** - Primary sources, receipts, evidence
2. **Respectful** - Dignity for all perspectives
3. **Hopeful** - Unity is possible
4. **Bold** - Unafraid to challenge narratives
5. **Human** - People over politics

### Do
- Present evidence, let readers conclude
- Acknowledge complexity
- Celebrate achievements on both sides
- Use "Korean people" over North/South divisions
- Focus on common heritage and future

### Don't
- Propaganda or inflammatory language
- Demonize any people (governments are fair game)
- Oversimplify geopolitics
- Ignore legitimate concerns
- Make promises you can't keep
